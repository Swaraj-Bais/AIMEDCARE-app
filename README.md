# 🩺 AIMEDCARE – AI Healthcare Assistant

AIMEDCARE is an AI-powered healthcare assistant designed to provide users with quick and reliable health-related guidance. It helps users understand symptoms, get medicine information, improve lifestyle habits, and promote overall wellness.

---

## 🚀 Features

- 🤖 AI Chatbot for healthcare queries  
- 💊 Medicine and symptom guidance  
- 🥗 Nutrition & diet suggestions  
- 😴 Sleep & stress management advice  
- ⚠️ Health awareness and preventive tips  
- 📱 Mobile-friendly (React Native based)

---

## 🛠️ Tech Stack

- **Frontend:** React Native (Expo)  
- **Backend / AI Integration:** Botpress Cloud  
- **Languages:** JavaScript / TypeScript  
- **Tools:** WebView, Expo Router

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Swaraj-Bais/AIMEDCARE-app.git
cd AIMEDCARE-app
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start the App (Expo)

```bash
npx expo start
```

---

## 📱 Run on Device

- Install **Expo Go** on your mobile  
- Scan the QR code from terminal  
- App will open instantly  

---

## 🔑 Configuration

- Ensure your Botpress WebChat script is correctly added  
- Update your bot ID / client ID in WebView if required
